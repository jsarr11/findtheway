Kruskal - οδηγός παιχνιδιού

Ο στόχος είναι να δημιουργήσετε το Ελάχιστο Καλυπτικό Δέντρο (MST) επιλέγοντας τις ακμές με τη μικρότερη τιμή χωρίς να δημιουργείτε κύκλους.

Βήματα για να παίξετε:

  1. Ξεκινήστε το παιχνίδι – Ο γράφος εμφανίζεται με κόμβους και ακμές που έχουν διαφορετικά βάρη.
  2. Επιλογή ακμών – Πατήστε στις ακμές με το μικρότερο βάρος πρώτα.
  3. Αποφυγή κύκλων – Αν μία ακμή δημιουργεί κύκλο, μην την επιλέξετε.
  4. Συνέχιση της διαδικασίας – Επιλέξτε ακμές μέχρι να έχετε Ν-1 ακμές, όπου Ν είναι ο αριθμός των κόμβων.
  5. Έλεγχος απάντησης – Πατήστε το κουμπί Έλεγχος για να ελέγξετε αν η λύση σας είναι σωστή.

Χρήσιμα κουμπιά στο παιχνίδι

  • Αναίρεση (Undo) – Μπορείτε να αφαιρέσετε την τελευταία ακμή που προσθέσατε είτε πατώντας το κουμπί αναίρεσης είτε επιλέγοντας μια ήδη επιλεγμένη ακμή.
  • Παύση (Pause) – Μπορείτε να διακόψετε το παιχνίδι προσωρινά.
  • Επανεκκίνηση (Start again) – Μπορείτε να ξεκινήσετε από την αρχή.
  • Σκορ (Scores) – Δείτε τις βαθμολογίες σας και των κορυφαίων παικτών.

Kruskal - game tutorial

The goal is to create the Minimum Spanning Tree (MST) by selecting the smallest-weight edges while avoiding cycles.

Steps to Play:

  1. Start the game – The graph appears with nodes and weighted edges.
  2. Select edges – Click on the edges with the lowest weight first.
  3. Avoid cycles – If an edge forms a cycle, do not select it.
  4. Continue – Keep selecting edges until you have exactly N-1 edges, where N is the number of nodes.
  5. Check your solution – Click Submit to verify your answer.

Useful Buttons in the Game

  • Undo – You can remove the last selected edge either by clicking the undo button or by selecting an already chosen edge.
  • Pause – Pause the game at any time.
  • Restart – Restart the game from the beginning.
  • Scores – View your score and top players' scores.
